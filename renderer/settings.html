Transcription engine

Built-in runs NVIDIA Parakeet privately, with nothing to install. OpenAI-compatible points at a local server, OpenAI, Groq, and the like.

Paste a Hugging Face URL or owner/model of a sherpa-onnx transducer bundle (encoder, decoder, joiner + tokens.txt). Custom models aren't checksum-verified.

The app calls {base URL}/audio/transcriptions.

Cleanup engine

Built-in runs a small Gemma model, with nothing to install. OpenAI-compatible points at Ollama, LM Studio, OpenAI, and the like.

Paste a Hugging Face GGUF model URL or owner/model. Earheart lists its versions and downloads the one you pick. Custom models aren't checksum-verified.

The app calls {base URL}/chat/completions.

    Performance

    Built-in models stay in memory for fast repeat dictations, then unload after this many idle minutes to free memory (~1.5 GB or more). Set to 0 to keep them loaded for the whole session. Only affects in-app models, not external services.

    Auto-paste permission

    Auto-paste types into the focused app, which macOS allows only with Accessibility permission. If pasting stopped working, click here to check the permission and open System Settings ▸ Privacy & Security ▸ Accessibility, then turn Earheart on (toggle it off and on again if it is already listed).
