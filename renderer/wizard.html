Welcome to Earheart

A couple of quick questions and you're dictating.

Earheart is private, hotkey-driven voice dictation. There is no account, no cloud requirement and nothing to learn — using it looks like this:

CtrlShiftSpace

1. Press the hotkey anywhere

→
0:03 ■

2. Speak, press it again

→
Your words appear.

3. Text lands where you type

This wizard pre-configures everything; you can review and change all of it on the settings screen afterwards.

Pick your dictation hotkey

One key combination toggles everything: press to start recording, press again to stop and transcribe.

The default works fine for most people — click the box to choose your own. On GNOME/KDE Wayland, apps may not be able to grab global keys — instead, bind a system keyboard shortcut to earheart --toggle.

Choose a microphone

The system default is usually right. Pick a specific device if you have more than one.

Where should speech become text?

Earheart ships a local server using NVIDIA's Parakeet model. Audio never leaves your machine.

Needs uv installed. The first start downloads the Parakeet model (~2.4 GB), so the very first transcription takes a while. See the stt-server README for details.

Any service that speaks the OpenAI audio transcription API works.

Optional — if the local server isn't installed yet, the test will fail and that's okay. You can test again from Settings later.

Tidy up transcripts? (optional)

A language model can fix punctuation and strip filler words ("um", "uh") after transcription. Works with any OpenAI-compatible chat API: Ollama, LM Studio, OpenAI, … Most people start without it.
