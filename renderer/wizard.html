Welcome to Earheart

A couple of quick questions and you're dictating.

Earheart is private, hotkey-driven voice dictation. There is no account, no cloud requirement and nothing to learn — using it looks like this:

CtrlShiftSpace

1. Press the hotkey anywhere

→
0:03 ■

2. Speak, press it again

→
Your words appear.

3. Text lands where you type

This wizard pre-configures everything; you can review and change all of it on the settings screen afterwards.

Pick your dictation hotkey

One key combination toggles everything: press to start recording, press again to stop and transcribe.

The default works fine for most people — click the box to choose your own. On GNOME/KDE Wayland, apps may not be able to grab global keys — instead, bind a system keyboard shortcut to earheart --toggle.

Choose a microphone

The system default is usually right. Pick a specific device if you have more than one.

Where should speech become text?

Earheart runs NVIDIA's Parakeet model directly inside the app. Your voice is transcribed on this computer — it never leaves your machine, there are no accounts or API keys, and it works offline.

Downloaded once on the next step.

Connect to a separately running Parakeet STT server (the earheart-stt package launched with uv).

Needs uv installed. The first start downloads the Parakeet model (~2.4 GB). See the stt-server README for details.

Any service that speaks the OpenAI audio transcription API works.

Tidy up transcripts? (optional)

A language model fixes punctuation, capitalization and strips filler words ("um", "uh") and false starts after transcription — so what lands in your document reads like writing, not a raw transcript.

Any GGUF chat model. Format: hf:<user>/<repo>/<file>.gguf or a direct download URL.

Set up your private models

Earheart is downloading the models it runs on this computer. This happens once. While it works, here's what you're getting:

  • Completely private — your voice and text are processed on your machine and never sent to a server.
  • Works offline — no internet needed after this.
  • Free, no accounts or API keys — nothing to sign up for, nothing to pay.

You can keep going through setup while this downloads — Earheart will finish the download in the background.
